# Flat Rate Tracker

A web app for automotive technicians to log repair orders and reconcile paid hours.

**Status:** Planning / Early Development
**Owner:** Liem

## The Problem
Flat rate technicians are paid per job completed, not per hour. Tracking what you've worked on, what got paid out, and whether the hours reconcile is a manual and error-prone process — and no good tool exists for it today.

## Phases

| Phase | Description | Target |
|-------|-------------|--------|
| 1 | Core app built and functional locally | In progress |
| 2 | Self-hosted on homelab, beta testers giving feedback | ~May 2026 |
| 3 | Hosted on a cloud provider, open to the public | Post-May 2026 |
